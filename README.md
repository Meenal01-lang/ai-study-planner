# 🎓 AI Study Planner

> Transform your study routine with AI-powered planning, smart scheduling, and productivity insights.

## 🚀 Overview

AI Study Planner is a full-stack web application that helps students create personalized study schedules, manage tasks, track progress, and stay exam-ready using Artificial Intelligence.

Instead of manually creating study plans, users simply enter their subjects, topics, exam dates, and available study hours. The AI then generates a structured study roadmap with revisions, priorities, and progress tracking.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/ai-study-planner.git
cd ai-study-planner
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
